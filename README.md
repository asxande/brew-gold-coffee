# Brew Gold Coffee

Cross-cultural academic project developed in partnership between students from Brazil and China.

## Objective

Prototype an e-commerce platform for introducing Brazilian specialty coffee to the Chinese market.

## Main Features

- Product Catalog
- Coffee Information Pages
- Shopping Cart
- Checkout System
- Product Reviews
- Customer Support
